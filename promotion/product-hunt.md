# Product Hunt Launch Copy

## Name

CodeKitBox

## Tagline

Private browser-based tools for everyday developer tasks

## Short description

CodeKitBox is a free multilingual toolkit for encoding, formatting, converting,
generating, and inspecting common developer values. Most tools run locally in
the browser, so inputs such as JSON, JWTs, and text stay on the user's device.

## Maker comment

Hi Product Hunt,

I built CodeKitBox because I often needed a small utility for one quick task:
formatting JSON, decoding a JWT, converting Unix time, generating a QR code, or
cleaning a list of text.

Many utility sites feel cluttered or require sending data to a server.
CodeKitBox is intentionally lightweight, requires no account, and processes
common inputs locally in the browser whenever possible.

The first release includes 19 tools, practical guides, and dedicated English,
Chinese, and Japanese pages. I would especially appreciate feedback on which
tools are most useful and what is still missing.

https://codekitbox.com/
