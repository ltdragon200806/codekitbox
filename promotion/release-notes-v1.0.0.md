# CodeKitBox v1.0.0

The first public release of CodeKitBox is now available.

CodeKitBox is a free multilingual collection of browser-based utilities. Common
text inputs are processed locally whenever possible, and no account is required.

## Highlights

- 19 practical developer and everyday tools
- English, Simplified Chinese, and Japanese tool pages
- Practical guides for Base64, JWT, JSON, QR codes, and strong passwords
- Clean URLs, sitemap, canonical links, hreflang, and structured data
- Static deployment with fast global delivery

## Popular tools

- JSON Formatter
- JWT Decoder
- Base64 Encoder and Decoder
- Unix Timestamp Converter
- QR Code Generator
- Strong Password Generator

Try it at **https://codekitbox.com/**
