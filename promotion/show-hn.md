# Show HN Draft

## Title

Show HN: CodeKitBox – private browser-based utilities with no account required

## Post

I built CodeKitBox, a static collection of small browser-based utilities:

https://codekitbox.com/

It currently includes 19 tools for tasks such as Base64 and URL encoding, JWT
decoding, JSON formatting, Unix timestamp conversion, UUID and QR code
generation, regex testing, text cleanup, and percentage calculations.

The main design constraint was that common text inputs should stay on the
user's device whenever possible. The site is plain HTML, CSS, and JavaScript,
with no account system and no application backend.

I also added dedicated English, Chinese, and Japanese pages plus practical
guides, because I wanted each utility to explain its limitations instead of
being only an input box.

I would appreciate feedback on the implementation, privacy wording, and which
tools deserve deeper functionality.
