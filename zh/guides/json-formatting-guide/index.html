如何格式化、验证和修复 JSON

标准 JSON 遵循严格的语法规则。格式化可以让嵌套数据更容易阅读，压缩则会删除不必要的空白。

打开免费工具

操作步骤

  1. 把 JSON 粘贴到格式化工具。
  2. 点击格式化，验证并添加缩进。
  3. 验证失败时查看错误位置。
  4. 修复引号、逗号、括号或属性名。
  5. 需要紧凑内容时点击压缩。

重要提示

  • JSON 字符串和属性名必须使用双引号。
  • 标准 JSON 不允许尾随逗号。
  • 格式化只改变空白，不会修改数据。

常见问题

为什么 JSON 无效？

常见原因包括尾随逗号、单引号和缺失括号。

压缩 JSON 会丢失数据吗？

不会，只会删除不必要的空白字符。
